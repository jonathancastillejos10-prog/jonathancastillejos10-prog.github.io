#### 🔍 Puntos Clave del Análisis:
* **Modelado de Cohortes:** Implementación de **CTEs** para rastrear el comportamiento de usuarios desde su fecha de registro.
* **Detección de Fugas:** Identificación de una caída del 40% en la retención durante el segundo mes de uso.
